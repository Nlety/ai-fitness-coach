## 项目简介

AI 健身教练是一款智能健身助手，提供个性化训练计划、动作指导和饮食建议。

## 核心功能

- 📋 **训练计划**：定制化健身方案
- 💪 **动作指导**：详细动作要领讲解
- 🥗 **饮食建议**：配合目标的饮食方案
- 💾 **训练记录**：保存训练历史

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
